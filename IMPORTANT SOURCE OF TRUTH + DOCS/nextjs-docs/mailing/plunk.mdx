---
title: Plunk
description: Learn how to use Plunk for sending emails.
---

To use Plunk with supastarter, first [create a Plunk account](https://useplunk.com/) and grab your API key.

Add the following environment variables to your `.env.local` file and your deployment environment:

```bash title=".env.local" copy
PLUNK_API_KEY="your-api-key"
```

Then, make sure to activate Plunk in the `/packages/mail/src/provider/index.ts`:

```typescript copy
export * from './plunk';
```

To customize the provider, edit the `packages/mail/src/provider/plunk/index.ts`.

For further instructions, follow the [general mail instructions](/docs/nextjs/mailing/overview).
