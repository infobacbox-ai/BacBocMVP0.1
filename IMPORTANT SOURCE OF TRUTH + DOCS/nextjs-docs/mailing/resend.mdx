---
title: Resend
description: Learn how to use Resend for sending emails.
---

To use Resend with supastarter, first [create a Resend account](https://resend.com/) and grab your API key.

Add the following environment variables to your `.env.local` file and your deployment environment:

```bash title=".env.local" copy
RESEND_API_KEY="your-api-key"
```

Then, make sure to activate Resend in the `/packages/mail/src/provider/index.ts`:

```typescript copy
export * from './resend';
```

For further instructions, follow the [general mail instructions](/docs/nextjs/mailing/overview).
